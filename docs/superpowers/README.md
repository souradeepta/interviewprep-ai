## Usage

1. **Review the Specification First**
   - Read the corresponding spec file (e.g., `2026-05-19-sde3-pattern-enhancement-design.md`)
   - Understand the requirements and approach

2. **Follow the Implementation Plan**
   - Use the plan file to execute step-by-step tasks
   - Check off completed tasks as you progress
   - Refer to templates for consistent formatting

3. **Track Progress**
   - Plans use checkbox syntax (`- [ ]`) for task tracking
   - Mark tasks as complete when finished
   - Commit work at natural breakpoints
